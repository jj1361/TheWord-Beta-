# Hebrew Letter Images

This directory should contain images for each Hebrew letter.

## Expected Image Files

Each image should be named according to the Hebrew letter:

- aleph.png
- bet.png
- gimel.png
- dalet.png
- hey.png
- vav.png
- zayin.png
- chet.png
- tet.png
- yod.png
- kaf.png
- kaf-final.png
- lamed.png
- mem.png
- mem-final.png
- nun.png
- nun-final.png
- samech.png
- ayin.png
- pey.png
- pey-final.png
- tzadi.png
- tzadi-final.png
- qof.png
- resh.png
- shin.png
- tav.png

## Image Specifications

- Format: PNG (recommended) or JPG
- Size: Recommended 300x300 pixels or larger
- Background: Transparent (for PNG) or white
- Content: Visual representation of the letter, its meaning, or symbolic representation

## Note

If images are not available, the application will gracefully hide the image section and still display all other letter information.
